# PRODIGY_WD_03 # ❌⭕ Tic Tac Toe Game

This project is a simple and interactive **Tic Tac Toe Game** built using HTML, CSS, and JavaScript.

---

## 🚀 Features

- Two-player game (X and O)  
- Interactive game board  
- Winner detection  
- Reset game functionality  
- Simple and clean UI  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript  

---

## ▶️ How to Run

1. Download or clone this repository  
2. Open the project folder  
3. Open `index.html` in your browser  

---

## 📂 Project Structure

tic-tac-toe/
│── index.html  
│── style.css  
│── script.js  
│── README.md  

---

## 🎯 Output

A fun and interactive Tic Tac Toe game where two players can play and see the winner.
